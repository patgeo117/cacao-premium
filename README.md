# Cacao Premium — Tienda Virtual ($1.000.000)

Tienda web de chocolate artesanal. Una sola página con catálogo, carrito y pedidos por WhatsApp.

**Demo:** https://cacao-premium.vercel.app

## Paquete incluido

- Página principal (inicio, productos, contacto)
- Catálogo con 5 productos
- Carrito con total e IVA
- Pedido por WhatsApp con mensaje prearmado
- Diseño responsive
- Despliegue en Vercel

## No incluye

- Panel administrativo
- Checkout con pasarela de pago
- Base de datos ni backend
- Dominio propio (paga el cliente)

## Desarrollo local

```bash
npm install
npm run dev
```
